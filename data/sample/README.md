# Sample data

Anonymized fixtures so the project is runnable without anyone's real spending data. Generated by
transforming a copy of real purchases: fake store names/CNPJs, jittered amounts, real structure and
categories. Loaded via `apps/api/scripts/import-legacy.ts` (or a dedicated seed script).

Real data NEVER lives here — it stays in the owner's Postgres instance. See
`docs/architecture.md#data--privacy`.
